nexus 6p刷安卓8

下载刷机包

解压

手机关机，先按住音量键下+电源键，进入reboot

进入解压后的刷机包，运行

等待过程，显示下图则成功

刷入SuperSU（Android 8）

连上数据线，开启开发者模式

运行命令

音量键向下按两次，进入recovery mode

向右划

选择install

进入Download 选择SuperSU

向右划

点击reboot system

向右划

手机就会重启

安装xpoder

安装ssr

“已连接，但无法访问互联网”开机后wifi有感叹号, 时间无法同步解决办法 在手机的shell里以root用户执行：

# settings put global captive_portal_http_url https://www.google.cn/generate_204

# settings put global captive_portal_https_url https://www.google.cn/generate_204

# settings put global ntp_server 1.hk.pool.ntp.org

# reboot

打开xposed，向下拉刷新

下载version 90

刷magisk（Android 10有效）

同样的方式刷入magisk，把magisk.zip放到sdcard根目录下

手机关机

先按住音量键下+电源键进入reboot

按两下音量键下，选择recovery mode

点击install

选择magisk.zip后，向右划

重新启动

刷nethunt

下载，或者window直接下载

把下载好后的nethunter放到sdcard下

按两下音量键下

选择nethunter

就开始自动刷机




点击NetHunter

菜单选择KeX Manager

点击SETUP LOCAL SERVER

设置密码



点击START SERVER开启服务器




连接手机

另一种 android8+magisk+LSPosed

https://developers.google.cn/android/images

https://topjohnwu.github.io/Magisk/install.html

android8+magisk+LSPosed

下载刷机系统包后，解压，进入，输入 ./flash-all.sh

安装magisk

修补镜像才能root

刷完后从刷机包里的镜像压缩包找到boot.img ，推送到 /sdcard/ 下

打开magisk，点击安装，选择并修补一个文件，选择刚才推送的文件，点开始，然后就开始patch了

显示All done后，adb shell进入到 /sdcard/Download/ 下，看到修补后的img，adb pull到电脑上

进入到解压的系统包，输入adb root bootloader 到刷机界面

输入 fastboot flash boot /path/to/magisk_patched.img （这里用自己修补后的）

刷完之后在输入 fastboot flash recovery /path/to/magisk_patched.img

重启之后就root成功了

下载 Riru 压缩包，push到手机/sdcard/，打开magisk点击模块，选择Riru，安装后

下载LSPosed，推送到 /sdcard/，，打开magisk点击模块，选择LSPosed，安装后重启
